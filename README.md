# Projects-Bazaar
Find Cool projects to work on and Collaborate with Others!
Projects Bazaar is a platform to find the coolest computer science projects 
to work on

You can also find potential collaborators by posting your project onto the platform within seconds

This project uses `pipenv` package manager

You can find all the dependencies inside  `Pipfile `
